HTML Nested Lists & Tables

What are Nested Lists?

Nested lists are lists inside other lists. They help create hierarchical structures where you can have main categories and subcategories. You can nest unordered lists (ul) inside ordered lists (ol) and vice versa.

Key Points:
  • Lists inside other lists create hierarchy
  • Can mix ordered and unordered lists
  • Nested lists must be inside <li> tags
  • Proper indentation makes code readable

Example 1: Basic Nested Unordered List

HTML Code:
<h3>Programming Languages</h3>
<ul>
  <li>Frontend
    <ul>
      <li>HTML</li>

HTML Table Theory

HTML tables are used to display data in rows and columns format. They organize information systematically, making it easy to compare and analyze data.

Basic Table Structure:
  • <table> - Main container for the entire table
  • <tr> - Table row (creates horizontal rows)
  • <td> - Table data cell (regular content)
  • <th> - Table header cell (bold and centered)
Advanced Table Elements:
  • <thead> - Groups header content (top section)
  • <tbody> - Groups body content (main data)
  • <tfoot> - Groups footer content (summary/totals)
  • <caption> - Provides table title or description
Important Table Attributes:
  • border="1" - Adds visible borders around cells
  • cellpadding="10" - Space inside cells (internal spacing)
  • cellspacing="5" - Space between cells
  • width="100%" - Sets table width
  • colspan="2" - Cell spans across multiple columns
  • rowspan="3" - Cell spans across multiple rows
  • align="center" - Aligns content (left, center, right)
  • bgcolor="#color" - Sets background color
Table Best Practices:
  • Use <th> for headers to make them bold and accessible
  • Structure with <thead>, <tbody>, <tfoot> for better organization
  • Use colspan and rowspan for complex layouts
  • Add borders for better readability
  • Keep table structure consistent

Example 1: Basic Simple Table

HTML Code:
<!DOCTYPE html>
<html>
<head>
  <title>Basic Table</title>
</head>
<body>

    <tr>
      <th>Name</th>
      <th>Age</th>
    </tr>
    <tr>
      <td>Souvik</td>

    </tr>
    <tr>
      <td>Riya</td>
      <td>21</td>
    </tr>
  </table>
</body>
</html>

Quiz - Nested Lists & Tables
1. What is the correct way to create a nested list?

Place <ul> directly inside another <ul>
Place <ul> or <ol> inside a <li> tag
Use the <nested> tag
Use special nested attributes

Check Answer Correct Answer: b) Place <ul> or <ol> inside a <li> tag
Nested lists are created by placing a new list (<ul> or <ol>) inside a <li> element of the parent list.
2. Which HTML tag creates table header cells?

<td>
<th>
<header>
<thead>

Check Answer Correct Answer: b) <th>
The <th> tag creates table header cells that are bold and centered by default.
3. What does the colspan="3" attribute do?

Creates 3 rows
Makes the cell span across 3 columns
Makes the cell span across 3 rows
Sets the cell width to 3 pixels

Check Answer Correct Answer: b) Makes the cell span across 3 columns
The colspan attribute makes a table cell span across multiple columns horizontally.
4. Can you mix <ul> and <ol> in nested lists?

No, you can only nest the same type
Yes, you can mix different list types
Only with special attributes
Only in newer browsers

Check Answer Correct Answer: b) Yes, you can mix different list types
You can nest ordered lists inside unordered lists and vice versa to create complex hierarchical structures.
5. Which tag groups the main content of a table?

<thead>
<tbody>
<tfoot>
<tcontent>

Check Answer Correct Answer: b) <tbody>
The <tbody> element groups the main data content of a table, separate from headers and footers.
6. What is the purpose of the <caption> tag?

To create table headers
To provide a title for the table
To add borders to the table
To create table cells

Check Answer Correct Answer: b) To provide a title for the table
The <caption> tag provides a title or description for the table and is placed right after the opening <table> tag.
7. What does rowspan="2" do to a table cell?

Makes the cell span across 2 columns
Makes the cell span across 2 rows
Creates 2 cells
Sets the cell height to 2 pixels

Check Answer Correct Answer: b) Makes the cell span across 2 rows
The rowspan attribute makes a table cell extend vertically across multiple rows.
8. Where should nested tables be placed?

Inside <tr> tags
Inside <td> or <th> tags
Before the <table> tag
Inside <thead> only

Check Answer Correct Answer: b) Inside <td> or <th> tags
Nested tables must be placed inside table cells (<td> or <th>) of the parent table.
9. What attribute controls the space inside table cells?

cellspacing
cellpadding
border
margin

Check Answer Correct Answer: b) cellpadding
The cellpadding attribute controls the space inside table cells (internal spacing), while cellspacing controls space between cells.
10. What is the correct order of table structural elements?

<tbody>, <thead>, <tfoot>
<thead>, <tbody>, <tfoot>
<tfoot>, <tbody>, <thead>
Order doesn't matter

Check Answer Correct Answer: b) <thead>, <tbody>, <tfoot>
The logical order is header first (<thead>), then main content (<tbody>), and finally footer (<tfoot>).
